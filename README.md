# PedroSilva

Website pessoal de Pedro Silva, focado na apresentacao do seu perfil profissional, experiencia em hotelaria e operacoes, formacao academica e contactos.

## Estrutura principal

- `index.html` - pagina inicial
- `about.html` - perfil e competencias
- `academica.html` - formacao, certificacoes e idiomas
- `profissional.html` - experiencia profissional e responsabilidades
- `contact.html` - contactos e acesso aos CVs
- `service-details.html` - areas de atuacao

## Assets principais

- `assets/css/main.css` - estilos base do template
- `assets/css/custom-styles.css` - personalizacao visual do projeto
- `assets/img/pedro-silva.jpg` - fotografia principal
- `assets/img/fundo.jpg` - imagem de fundo usada no hero
- `assets/files/pedro-silva-cv-2026.pdf` - CV atualizado
- `assets/files/pedro-silva-cv-europass.pdf` - CV Europass

## Notas

- O projeto foi limpo para remover referencias antigas a Goncalo Garrido e aos respetivos projetos.
- O tema visual atual usa tons de azul escuro.
- A tipografia principal usa `Source Sans 3` com headings em `Merriweather`.
- As paginas sao estaticas e podem ser abertas diretamente no browser.
